How to host this yourself
  1. Create a free GitHub account at github.com
  2. Create a new repository called brain-dump-dashboard
  3. Upload this HTML file to the repo
  4. Go to Settings → Pages → Source → main branch → Save
  5. Your app will be live at: https://[yourusername].github.io/brain-dump-dashboard
  6. Bookmark that URL — it's yours forever
